Blockland Tower

You must install everything here to start playing.
Note: Blockland Tower isn't public yet. Feel free to look at the client code, if you want. Don't use any assets as they might be better soon.
Fonts

Blockland Tower requires these fonts for your client to work correctly.

Mac Extract the .TTF files to Library/Fonts.
Windows Extract the .TTF files anywhere, select them all, right click, Install.

Add-On

Simply place the Client_Blockland_Tower.zip file in Blockland\Add-Ons. Then, relaunch the game and join the tower. This is not a server Add-On; it does not allow you to host your own Blockland Tower game. It's a required Client Add-On that allows you to play on the official server.
